方法的重载

  • 重载就是在一个类中，有相同的函数名称，但形参不同的函数。

  • 方法的重载的规则:

    • 方法名称必须相同。

    • 参数列表必须不同(个数不同、或类型不同、参数排列顺序不同等)。

    • 方法的返回类型可以相同也可以不相同。仅仅返回类型不同不足以成为方法的重载。

  • 实现理论:

    • 方法名称相同时,编译器会根据调用方法的参数个数、参数类型等去逐个匹配，以选择对
      应的方法，如果匹配失败，则编译器报错。
