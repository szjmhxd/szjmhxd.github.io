编译型和解释型

有一个只会英语的美国人想看《三体》

  • 编译型 买一本英文的《三体》
  • 解释型 请了一个翻译一句一句讲给他听

  • 编译型

    1. 执行速度更快 已经编译好了 电脑直接执行即可

    2. 操作系统会用编译型语言

    3. c c++ 是编译型语言

  • 解释型

    1. 网页或脚本对运行速度要求不高

Java 虚拟机的工作原理

  • Java源程序(*.java)经过Java编译器的预编译，编译成字节码(*.class)文件

  • 字节码在运行的时候到 JVM虚拟机的类装载器

  • 虚拟机中有一个字节码校验器，检测代码的合法性和安全性

  • 如果代码正确，去解释器解释执行代码

随着技术和硬件的发展，编译型和解释型的界限越来越模糊（内存越来越大，CPU越来越快，解释型语言的运行速度快了）
