使用Docker部署RustDesk服务器

RustDesk是一款开源远程桌面软件，本文将详细介绍如何使用Docker部署RustDesk服务器及其API服务。

一、准备工作

1. 环境要求

  • Docker 和 Docker Compose
  • 服务器一台
  • 域名或公网IP地址

2. 相关资源

  • RustDesk API项目：lejianwen/rustdesk-api (官方免费版不支持web api，无法登录和管理)
  • 官方文档：Use Docker With Rustdesk-Server-S6
